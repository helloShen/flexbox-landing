Flexbox layout is awesome
Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex to fill additional space or shrink to fit into smaller spaces.
Learn now
Why flexbox layout?
Flexbox is a one-dimensional layout method.
The flex-direction property allows us to change the direction in which our flex items display.
An area of a document laid out using flexbox is called a flex container.
Grid layout is an alternative to flexbox.
